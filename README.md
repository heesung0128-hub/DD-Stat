# DD Stat 공유 및 다른 컴퓨터 실행 가이드 (동덕여고 수학과)

본 프로그램은 서버 통신이 필요 없는 **순수 클라이언트 사이드 웹 어플리케이션**입니다. 따라서 USB, 이메일, 메신저 등으로 파일만 전달하거나, 인터넷 링크(URL) 형태로 아주 손쉽게 배포할 수 있습니다. 

공유하고 활용하는 두 가지 방법을 아래와 같이 안내합니다.

## 방법 1. 압축 파일(ZIP)로 전달하여 로컬에서 실행하기 (가장 간단함)

프로젝트 폴더 내에 생성된 **[DD_Stat_Dongduk.zip](file:///C:/Users/user/.gemini/antigravity/scratch/highschool-stats/DD_Stat_Dongduk.zip)** 파일을 다른 사람에게 메일, 카카오톡, USB 등으로 공유합니다.

### 💻 받는 사람의 실행 방법:
1. 전달받은 `DD_Stat_Dongduk.zip` 파일의 압축을 풉니다.
2. 압축이 풀린 폴더 내에서 **`index.html`** 파일을 마우스 더블클릭합니다.
3. 브라우저(크롬, 엣지, 웨일 등)가 열리며 즉시 실행됩니다.
   * *주의: 통계 연산 및 그래프 그리기를 위한 외부 라이브러리를 CDN에서 실시간으로 가져오므로, 인터넷이 연결된 환경이어야 합니다.*

## 방법 2. 웹 사이트(URL 링크)로 무료 배포하기 (추천)

사용자가 파일을 내려받을 필요 없이, 인터넷 주소(예: `https://아이디.github.io/dd-stat`)만 입력하면 PC, 태블릿, 스마트폰 등 어디서나 접속해 사용할 수 있게 만드는 방법입니다.

정적 페이지이므로 **GitHub Pages** 또는 **Netlify**를 통해 **3분 만에 무료로 배포**할 수 있습니다.

### 🚀 GitHub Pages를 통한 초간단 배포 절차:
1. [GitHub](https://github.com)에 로그인(계정이 없다면 가입)합니다.
2. 우측 상단의 `+` 버튼을 누르고 **New repository**를 클릭합니다.
   * **Repository name:** `dd-stat` (또는 원하는 이름)
   * **Public**으로 설정하고 최하단의 **Create repository**를 누릅니다.
3. 생성된 빈 저장소 화면 중앙 of **"uploading an existing file"** 링크를 누릅니다.
4. 압축을 푼 폴더 내부의 파일들(**`index.html`, `style.css`, `stats-helper.js`, `app.js`**)을 드래그 앤 드롭하여 업로드합니다.
5. 화면 하단의 녹색 **Commit changes** 버튼을 누릅니다.
6. 저장소 상단 메뉴의 **Settings**(설정) ➡️ 좌측 메뉴의 **Pages**로 이동합니다.
7. Build and deployment ➡️ Branch 부분의 **None**을 **`main`**(또는 `master`)으로 변경하고, 바로 옆의 **Save** 버튼을 누릅니다.
8. 1~2분 정도 대기 후 페이지를 새로고침하면 상단에 **`Your site is live at https://[내-아이디].github.io/dd-stat/`** 형태의 전용 인터넷 주소가 발급됩니다. 이 주소를 공유하시면 됩니다!

### ⚡ Netlify를 통한 초간단 배포 절차 (가장 빠름):
1. [Netlify Drop](https://app.netlify.com/drop) 사이트에 접속합니다 (회원가입 불필요).
2. 화면 중앙의 드래그 구역에 압축을 푼 폴더(또는 공유할 4개 파일)를 그대로 드래그해서 올려놓습니다.
3. 업로드 즉시 임의로 발급된 무료 사이트 주소(예: `https://xxxxxx.netlify.app`)가 나타납니다. 이 주소를 친구들과 즉시 공유하시면 됩니다!
